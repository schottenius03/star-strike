# 🚀 StarStrike

**Project Overview:** A 3D space arcade action game built within Unreal Engine, focusing on visual scripting, mechanics design, and interactive 3D environments.  
**Tech Stack:** Unreal Engine, Blueprints, Visual Scripting

## 📌 Overview

StarStrike is a desktop game application created to explore interactive mechanics and logic design within a 3D simulation environment. The project leverages Unreal Engine Blueprints to connect game events, manage actor behaviors, handle projectile velocity, and drive dynamic gameplay loops. By utilizing node based visual logic, the project demonstrates rapid prototype design and modular feature implementation.

## Core Features

* **Visual Logic Systems:** Built game systems using visual blueprints to track player lives, handle weapon configurations, and trigger enemy spawning.
* **Physics & Interactions:** Configured collision zones and movement vectors for space flight and projectile targets.
* **Engine Framework:** Utilized standard Unreal Engine components to manage game states, user interfaces, and audio cues during action sequences.

## Gameplay Mechanics

* **Flight Controls:** Setup custom input mapping for responsive and fluid space navigation.
* **Wave System:** Created an enemy wave structure designed to challenge player coordination.
* **User Interface:** Designed visual menus and overlay displays to track player scoring milestones in real time.
